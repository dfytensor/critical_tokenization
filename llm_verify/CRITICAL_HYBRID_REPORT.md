# Critical Generative Modeling（临界生成建模）hybrid 验证 —— 🟡 部分成立

针对"能否围绕 γ/长程结构重组训练范式"的蓝图，验证最现实的那条：**hybrid loss = CE + 互信息曲线匹配（REINFORCE）**。结果：**比纯 γ-loss 好得多——结构项确实方向性可控，但弱、不对称、且以连贯性为代价。**

## 实验设计（关键改进 vs 失败的 γ-REINFORCE）
- **匹配整条 I(d) 曲线**（d=1..8），不是 γ 单标量 → reward 量级 ~1-10 bit²（vs γ-only 的 ~0.001），信号强 3 个量级。
- **CE 锚**（每步在真实 chosen 序列上做 CE）→ 保住连贯区，γ 信号才有意义。
- 三对照（同 SFT 基线，100 步）：control(纯CE) / low(target I×1.5，更临界) / high(target I×0.5，退化)。
- 人类 I(d) 基线：[5.85, 4.94, 4.8, 4.69, 4.58, 4.59, 4.59, 4.59] bit。

## 结果

| condition | 起 γ | 终 γ | 终 rep4 |
|---|---|---|---|
| control（纯 CE）| 0.047 | **0.028** | 0.143 |
| low_gamma（I×1.5）| 0.047 | 0.043 | 0.127 |
| **high_gamma（I×0.5）**| 0.047 | **0.072** | **0.257** |

**核心发现：结构项产生了区分于纯 CE 的方向性影响**——
- high_gamma 把 γ 推到 **0.072**（最高），control 反而掉到 0.028 → **结构项能向上拉 γ**。
- low_gamma 在 control 一路下滑时**守住**了 γ（0.043 vs control 0.028）。
- **CE 锚奏效**：rep4 没崩（0.127-0.257），CE loss 维持在 ~3.0，模型没退化成乱码。

→ **对照之前失败的纯 γ-REINFORCE**（两目标走出同一条 γ 轨迹，完全无控）：hybrid 用"曲线匹配 + CE 锚"两招，**把 γ 从"不可驱动"变成"方向性可驱动"**。蓝图的核心直觉被验证。

## 诚实边界（为什么只是"部分"）
1. **弱**：γ 只移动了 ~0.04（0.028→0.072），幅度小。
2. **不对称**："降低 MI（high direction）"容易（γ↑明显），"增加 MI（low direction）"难（几乎只是守住了 control 的下滑）——**破坏长程结构容易、重建难**，符合物理直觉（熵增易、自组织难）。
3. **以连贯性为代价**：high_gamma 的 rep4 升到 0.257（更重复）——拿 γ 当目标会把模型推向"局部高概率、长程失联"。
4. 玩具规模（60M，100 步，T0.8 区 γ~0.05）；生产级未测。

## 对原问题的最终回答（现在三层都有实测）
| 设想 | 实测 |
|---|---|
| γ 当监控 | ✅ 真 DPO 确认 |
| γ 当纯 loss（γ-REINFORCE）| ❌ 不可控（Task 1）|
| γ-aware 解码（闭环 T）| ❌ 不可控（Task 2）|
| **hybrid：CE + MI 曲线匹配** | **🟡 方向性可控**（曲线匹配+CE 锚比 γ-only 强得多）|

**净结论**：蓝图的方向**不是空想**——把"整条 I(d) 曲线"（而非 γ 单标量）匹配 + CE 锚定，确实让结构项第一次产生了可辨识的方向性控制。但它**弱、不对称、伤连贯性**，离"治愈长程漂移"还远。这是"临界生成建模"作为**微调阶段的结构对齐正则**（类似 RLHF，奖励是结构而非偏好）的**可行性证据**，不是替代 CE/自回归的成熟方案。

下一步要把它做实，需要：更长序列稳定 I(d) + reward reshaping（治不对称）+ 多矩匹配（不止 I(d)，加块熵/谱）+ 更低方差估计器（CPC 式对比 MI）。这些都是真研究问题，但**第一盏灯（曲线匹配+CE 锚可行）已被点亮**。
