# 闭环验证 1：高阶矩（块熵）治退化 —— ✅ 理论预测证实

承接"理论与实验焊在一起"的纲领。理论（RG/统计简并）对 hybrid 实验的"伤连贯(rep4↑)"给出的预测：**单一 I(d) 统计量有简并自由度，模型会用重复作弊；加高阶统计量（块熵 H(ℓ)）能提升可辨识性、压制退化。** 这里直接测这个预测。

## 设计（最可证伪的那条）
两条件，都跑 high_gamma 方向（I×0.5，基础 hybrid 里 γ 可控但 rep4↑ 的那条）：
- **v0**：CE + L2(I(d), target=0.5×I_data) —— 基础 hybrid（应重现 γ 控 + rep4↑ 退化）。
- **v1**：CE + L2(I(d), 0.5×I_data) **+ L2(H(ℓ), target=H_data)** —— 加块熵匹配，逼模型不能用重复（重复会让 H(ℓ) 塌缩）来满足 I(d)。

人类基线：I(d)=[5.85,4.94,...] bit；H(ℓ)=[9.49,13.13,14.56,15.01,15.14,15.2] bit。两曲线分别按各自尺度归一（I_scale=23.5, H_scale=193）。

## 结果

| condition | γ 终 | rep4 终 | i_term |
|---|---|---|---|
| v0（无 H 项）| 0.045 | **0.209↑**（退化）| ~0.46 |
| **v1（+块熵）**| 0.013 | **0.147（持平）**| ~0.39 |

**核心：v1 的 rep4(0.147) 明显低于 v0(0.209)，且没从基线 0.158 上升** → **高阶矩（块熵）压制了基础 MI-matching 的重复退化**，正是理论预测。两个条件下 I(d) 匹配项 (i_term) 量级相当（0.4-0.5），说明 H 项不是靠"挤掉 I 项"起作用，而是单独堵住了退化解。

## 闭环成立
- 理论（统计简并 / RG）→ 预测：单一统计量会引来退化解（重复），高阶矩可治。
- 实验：v0 重现 rep4↑；v1 把 rep4 压住。**预测被证实。**
- → 这是"理论解释实验 → 修正理论 → 实验反哺"的第一圈闭环，且方向正确。

## 诚实边界
1. v1 的 γ 没去到 high_gamma 目标（0.013，反而更"临界"）——因为 H 项(target=高熵人类)与 I×0.5(低 MI) 方向冲突，净效果是 γ 下行。说明**多目标要仔细加权**（理论里"临界流形"是多个标度关系的耦合，不能简单线性相加）。
2. rep4 差异虽清楚（0.147 vs 0.209）但幅度不大；玩具规模（60M/100步）。
3. **理论修正的另两条还没测**：(a) 多尺度/MMD 治"不对称"；(b) 有限尺度标度治"信号弱"。本实验只验了高阶矩治退化这一条。

## 下一步（闭环继续）
- 测"不对称"：把 L2 换成 **log 域（重斜率 γ）+ 多 lag 匹配**，看 low/high 两方向是否更对称可控。
- 测"信号弱"：**有限尺度标度**——用多个长度 L 的序列联合拟合 I_L(d) 标度函数，看信号 Δγ 能否从噪声中放大。
- 这两条若也成立，"临界流形上的结构对齐"就从**单点验证**升级为**系统性可行**。
