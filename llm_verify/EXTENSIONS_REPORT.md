# 扩展实验报告：⑤ 可微词器代理 / ⑥ 跨语言 / ③ γ 监控（60M 连贯模型）

按"三个都要"做的最小可行版本，各自老实标注。

## ⑤ 渐进式扩词（可微词器的可证伪近似）— ❌ 比 abrupt 更差

离散动态分词（abrupt 切换）之前已证伪。可微词器的动机是"消除切换扰动"，用**渐进扩词**（多次小切换）做最简代理测试。A→B 漂移，500 步：

| 条件 | B-val BPC |
|---|---|
| frozen（m1500 贯穿）| **1.980（最好）** |
| oracle（full 贯穿）| 2.001 |
| abrupt（A=m1500, B=full 一次切）| 2.060 |
| **gradual（A=m1500, B 分 4 段渐进扩到 full）** | **2.113（最差）** |

→ **gradual 比 abrupt 更差**：4 次切换比 1 次更扰动。问题不是"切换太突然"，而是**每次扩词都引入未训练的 embedding 行；切换越多越乱**。可微词器的"平滑"假设在最简代理下**不被支持**。frozen>oracle 还印证：欠训练下小词表反而赢（embedding 税）。

**唯一出路**：真·连续软分词（Gumbel/VQ-VAE，无任何离散切换）——本次未实现，是下一篇的工作。

## ⑥ 跨语言 α≈1 普适性 — 🟡 方向对，数据拖后腿

| 语言 | 文字 | char α | BPE V\* |
|---|---|---|---|
| en | 拉丁 | — | ~1500（既有）|
| de | 拉丁 | 1.155 | 3590 |
| fr | 拉丁 | 1.068 | 2128 |
| ru | 西里尔 | 1.318 | 1113 |
| cn | CJK | 1.057 | ~12k 字符（既有）|
| es/it/pt | 拉丁 | 0.85-0.90 | **不可靠**（抓到纯 ASCII 文本，退化）|

→ **干净信号**：字母/西里尔语 BPE 临界 V\*≈1-4k，CJK≈12k 字符——**临界点普适，临界单位随文字变**（字母=子词，表意=字）。但 es/it/pt 数据退化、de/ru 样本小，普适性"方向"成立、未坐实。

## ③ γ 监控（60M 连贯模型）— 🟡 弱阳性（从 8M 的❌升级）

之前 8M 玩具生成乱码、γ 基线无意义（漂移仅 +0.02）。改用 **60M SFT 连贯模型（67.5M 参数，中文）**：

| | γ | distinct-2 | rep4 |
|---|---|---|---|
| **人类（真 token）** | **0.147** | 0.352 | 0.179 |
| 打乱 | 0.0005 | — | — |
| 连贯 60M baseline（greedy / T0.8）| 0.118 / 0.095 | 0.10 / 0.36 | 0.75 / 0.27 |
| 过拟合退化后（greedy / T0.8）| **0.174** / 0.118 | 0.26 / 0.42 | 0.37 / 0.16 |

**两个发现**：
1. 连贯模型 γ（0.10-0.12）与人类（0.147）**同量级**——基线有意义了（不像 8M 玩具）。
2. **γ 与局部指标在退化时吵架**：过拟合后 distinct-2 升、rep4 降（局部指标说"更多样=更好"），但 **γ 升（长程连贯性流失=真退化）**。γ 抓到了局部指标误读的盲点。greedy γ 漂移 +0.056 较清楚，T0.8 +0.023 较弱。

→ γ 监控从"8M 的❌"升级到"60M 的🟡弱阳性"：**方向对、补局部指标盲点，但信号幅度小（~0.02-0.06）**。要坐实仍需真 DPO（奖励过优化谱），不是过拟合代理。

## 本轮（②③④⑤⑥）总结

| | 结果 |
|---|---|
| ② 多种子 V-N | ✅ 0.666±0.000（精密，非噪声限）|
| ④ 中文真 V\* | ✅ ~12k 字符，α≈1 规模稳健 |
| ⑥ 跨语言 | 🟡 方向对（单位随文字变），数据不足 |
| ⑤ 可微词器代理 | ❌ 渐进比离散更差 |

**研究硬核坐实**：临界结构、α≈1、V-N 小规模律、γ 跟采样退化。
**边界反复撞墙**：动态>静态、γ 监控真实退化、大 N 律——在可及规模下都"方向对但没坐实或证伪"。下一步唯一实质方向：**真上规模（1B+ Chinchilla）+ 真 DPO 管线**。
